# A Beautiful Game KTX2 + Draco Fixture

`abeautifulgame-ktx-draco.glb` is derived from the Khronos glTF Sample Assets
`Models/ABeautifulGame/glTF-Binary-KTX-ETC1S-Draco/ABeautifulGame.glb`
fixture.

Source:
https://github.com/KhronosGroup/glTF-Sample-Assets/tree/main/Models/ABeautifulGame

The committed Aperture fixture keeps the Khronos sample geometry, Draco mesh
compression, and embedded KTX2/BasisU texture payloads. Optional
`KHR_materials_transmission` and `KHR_materials_volume` material-extension
metadata was removed so the sample exercises Aperture's current
StandardMaterial-compatible import path without claiming support for those
future material extensions.

Original model: MaterialX Project / ASWF, 2020. Conversion to glTF: Ed Mackey, 2022. License: CC BY 4.0.
