# Basis Universal Transcoder Assets

This directory contains the Basis Universal transcoder JavaScript and WASM files
used by the local examples to decode `KHR_texture_basisu` / KTX2 textures.

The files were copied from `references/three.js/examples/jsm/libs/basis/` so the
examples can run without network fetches. Keep them treated as vendored generated
assets; update them deliberately as a pair when the reference copy changes.
