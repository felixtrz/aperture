# Kenney Particle Pack sample textures

Sources:

- https://www.kenney.nl/assets/particle-pack
- Kenney racing road tile, copied from the local racing showcase assets for
  this particle scene.

The selected PNG files in this directory are from Kenney Particle Pack 1.1,
licensed Creative Commons Zero (CC0). See `LICENSE.txt`.

Only the particle textures, one small ground model, and its `Textures/colormap.png`
dependency used by the Aperture particle showcase example are included here to
keep the example payload small.
