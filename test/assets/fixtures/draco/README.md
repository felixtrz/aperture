# Draco Decoder Fixtures

These fixtures support the public Draco decoder tests.

- `bunny.drc` is copied from
  `references/three.js/examples/models/draco/bunny.drc`.
- `draco_wasm_wrapper.js` and `draco_decoder.wasm` are copied from
  `references/three.js/examples/jsm/libs/draco/gltf/`.
- `heart_draco.glb` is copied from
  `references/engine/examples/assets/models/heart_draco.glb`.

Keep the wrapper and WASM files updated together.
